### Currency Analysis Tool
This is a stage-based python project, it begins as a simple GBP currency conveter and gradually evolves into a flexible(able to handle growth), adaptable currency analysis platform. 
This project is part of my software engineering learning journey, it demonstrates my progression in a clear format, and my continuous enhancement and development across multiple stages. 

## Project Overview
The goal of this project is to build a currency conversion and analysis tool step‑by‑step, improving it’s:

- code quality
  
- structure
  
- scalability
  
- user experience
  
- engineering thinking

Each stage is stored in its own folder to clearly demonstrate growth and professional development.

## Stages
