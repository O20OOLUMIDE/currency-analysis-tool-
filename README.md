# Currency Analysis Tool

A stage‑based Python project that begins as a simple GBP currency converter and gradually evolves into a more advanced, scalable currency analysis system.  
This project is part of my learning journey and software engineering portfolio, showing clear progression, refactoring, and feature expansion across multiple stages.

---

##  Project Overview

The goal of this project is to build a currency conversion and analysis tool step‑by‑step, improving code quality, structure, and functionality at each stage.  
Each stage is stored in its own folder to clearly demonstrate growth and engineering thinking.

---

##  Stages

### **Stage 1 — Basic GBP Converter**
- Converts GBP → USD or EUR  
- Simple input handling  
- Basic loop structure  
- First working version of the tool  

**Folder:** `stage1_basic_converter/`

---

### **Stage 2 — Extended Multi‑Currency Converter**
- Added support for USD, EUR, and JPY  
- Improved input validation  
- Added positive‑number checks  
- Introduced a conversion counter  
- Cleaner user prompts and error messages  
- More robust and user‑friendly structure  

**Folder:** `stage2_extended_converter/`
